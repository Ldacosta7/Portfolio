Poké API

Technologies : HTML/CSS + JS + API (https://pokeapi.co/)

Durée du projet : En cours

Contexte : Apprentissage de l'utilisation d'une API avec pour sujet le Pokédex. Le site permet actuellement de génerer le sprite d'un pokemon au hasard et d'afficher son nom et son numéro dans le pokédex. De plus un bouton permet de modifier le sprite en passante du sprite normal au sprite shiny.

Modification à venir : Le site affichera en plus des infos existante les statistiques ainsi que le Movepool du pokémon généré.
