Mes réalisations

Mes projets scolaire

Disneyland - Site Web

Technologies : HTML/CSS + PHP + MySQL suivant le modèle MVC

Durée du projet : 2 mois

Contexte : Réalisation de la refonte du système de billeterie de Disneyland Paris. Le site permettra au client de créer un séjour personnalisé dans le parc tandis que les employé du parc aurons différents droit leur permettant l’ajout ou la modification d’attraction, de restaurants etc.

Disneyland - Logiciel de Support

Technologies : Java (Swing/awt)

Durée du projet : 1 mois

Contexte : Réalisation d'un logiciel de support pour les employé de Disneyland Paris. Le logiciel doit permettre aux employés qualifiés de se connecter et d’avoir accès a des données qu’il pourront ajouter, modifier ou supprimer.

Disneyland - Application Mobile

Technologies : Android Studio

Durée du projet : 2 semaines

Contexte : Réalisation d’une application permettant d’interroger la clientèle du parc. L'application doit permettre aux utilisateur de répondre à un questionnaire de satisfaction sur différents sujets et les réponses doivent pouvoir être afficher dans l’application.
